# Spatial Computing Lab

An interactive web-based AR platform for visualizing electronics and embedded systems experiments using 3D models and QR codes.

## Features
- Interactive 3D circuit models
- Fullscreen & AR support
- Experiment-wise circuit breakdown
- Designed for E-Yantra IoT & Embedded Systems Lab

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- model-viewer (WebXR)

## Usage
Scan QR codes in the lab or browse experiments directly.
